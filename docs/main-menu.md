# Main Menu

Send `/menu` at any time to open the Polytragent main menu. The menu has 5 sections, each as a full-width button:

```
🔬 Event Research     ← Top button, full width
📊 Portfolio
📈 Strategies
🔬 Research
⚙️ Settings
```

## Menu Overview

| Button | What It Does |
|--------|-------------|
| 🔬 **Event Research** | Paste a Polymarket event link for full AI analysis — market data, recommendation, whale activity, news context, entry/exit strategy |
| 📊 **Portfolio** | View your positions, P&L, risk scorecard, closed trades, and performance analytics |
| 📈 **Strategies** | AI strategy signals — NO Theta Decay and Scalping NO Theta with entry gates, exit rules, and position sizing |
| 🔬 **Research** | Market intelligence — trending events, new markets, whale alerts, breaking news, global stats |
| ⚙️ **Settings** | Wallet connection, bet sizing, notifications, risk limits, API keys, data export |

## Navigation

Every sub-menu has a **← Main Menu** button to return to the top level. Sub-sections also have back buttons to their parent menu (e.g., **← Portfolio**, **← Research**, **← Strategies**).

## Quick Tip: Auto-Research

You can paste a Polymarket event link **anywhere in the chat** at any time — the bot automatically detects it and runs Event Research. No need to navigate to the menu first.
