# Polytragent

## Your AI-Powered Polymarket Trading Agent

Polytragent is a Telegram-based AI trading agent for Polymarket — the world's largest prediction market. It provides real-time event research, portfolio tracking, AI strategy signals, whale alerts, and breaking news — all inside Telegram.

### What You Get

- **Event Research** — Paste any Polymarket link, get full AI analysis with market data, whale activity, and entry/exit strategy
- **Portfolio Management** — Connect your wallet for live positions, P&L tracking, risk scoring, and performance analytics
- **AI Strategy Signals** — Pre-built NO Theta and Scalping strategies with proven entry gates and risk parameters
- **Whale Tracking** — Follow top wallets, get push notifications when whales trade, monthly P&L leaderboard
- **Live Market Data** — Trending events by 24h volume, new markets, breaking news, and global Polymarket stats

### Quick Start

1. Open Telegram and search for **@bmpolybot**
2. Send `/menu` to open the main menu
3. Subscribe for $99/mo or enter an access code
4. Connect your Polymarket wallet in Settings
5. Start researching events and trading
